# Ejercicio 01 - Administración básica en Windows Server 2022


## Objetivo

Realizar tareas básicas de administración en un servidor Windows,
incluyendo información del sistema, usuarios, grupos, servicios y
uso básico de PowerShell.

## Entorno

- Sistema: Windows Server 2022 Standard (Desktop Experience)
- Virtualización: VirtualBox
- Nombre del servidor: SERVER01
- Usuario administrador: Administrador

## Tareas realizadas

- Comprobación de información del sistema
- Gestión de usuarios locales
- Gestión de grupos locales
- Administración de servicios
- Uso básico de PowerShell
